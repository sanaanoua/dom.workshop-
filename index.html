4. Respond to a click event

Right at the bottom of this div, you'll find a #open-modal button, and a modal (with #modal as its id) which contains a #close-modal button.

Get the two button and the modal div elements. Then create two event listeners — one for each button:

  • When the #open-modal button is clicked, the modal should be shown, by manipulating its display style property (initially set to none in the CSS).
  • Conversely, when the #close-modal button is clicked, the modal should be hidden.
